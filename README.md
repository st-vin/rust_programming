# Rust Learning Projects

A collection of Rust projects for learning core language concepts and features.

## Projects

### hello_world
Basic Rust program demonstrating simple output. Entry point for Rust development.

### hello_cargo
First Cargo project setup. Introduction to Rust's package manager and build system.

### variables
Variable declaration, mutability, and shadowing. Understanding Rust's approach to mutable and immutable bindings.

### data_types
Scalar and compound types including integers, floats, booleans, characters, arrays, and tuples. Numeric operations and type inference.

### control_flow
Conditional expressions with if/else, loops (for, while), loop labels, and control flow patterns.

### functions
Function declarations, parameters, return values, and expression-based returns.

## Getting Started

Each project is a standalone Cargo package. To build and run any project:

```
cd project_name
cargo build
cargo run
```

## Requirements

Rust toolchain installed. Download from https://www.rust-lang.org/
